Tasks:
==================
 01) Пользователь вводит значения b,c (катеты), программа возвращает значения гипотенузы
 02) Пользователь вводит значение стороны правильного треугольника, возвращается площадь
 03) Пользователь вводит значение оснований и высоты трапеции, возвращается площадь
 04) Пользователь вводит значение строны квадрата, возвращается площадь
 05) Пользователь вводит значение строны и высоты параллелограмма, возвращается площадь
 06) Пользователь вводит значение двух сторон параллелограмма и угол между ними, возвращается площадь
 07) Пользователь вводит радиус круга, возвращается площадь
 08) Пользователь вводит радиус сферы, возвращается площадь
 09) Пользователь вводит радиус шара, возвращается объем
 10) Пользователь вводит радиус высоту и радиус цилиндра, возвращается объем
 11) Пользователь вводит расстояние и время, возвращается скорость
 12) Пользователь вводит начальную скорость, время и ускорение, вычисляется расстояние
 13) Пользователь вводит линейную скорость и радиус, вычисляется центростремительное ускорение
 14) Пользователь вводит массу и силу, вычисляется ускорение (по второму закону Ньютона)
 15) Пользователь вводит массу, возвращается вес (на Земле)
 16) Пользователь вводит массу, возвращается вес (на Луне)
 17) Пользователь вводит массу и объем, вычисляетс плотность
 18) Пользователь вводит глубину. Программа вычисляет давление воды на этой глубине.
 19) Пользователь вводит напряжение и сопротивление, вычисляется сила тока
 20) Пользователь вводит напряжение и сопротивление, вычисляется мощность
 21) Пользователь вводит фамилию и имя, программа выводит сообщение, например: "Привет, Иван Петров!"
